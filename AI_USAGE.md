\# AI Usage



\## Where I Used AI

I used AI tools primarily for getting an initial structure of the HTML layout and some basic CSS styling ideas. This helped me quickly set up the UI and focus more on the functionality of the application.



\---



\## What I Implemented on My Own

The core logic of the application was implemented by me. This includes:

\- Service listing and booking functionality

\- Handling user inputs and storing data using local storage

\- Designing and implementing the AI-based recommendation system



I independently worked on the logic to interpret user input, match it with relevant services, and generate meaningful recommendations.



\---



\## GenAI / RAG Understanding

I applied my own understanding of GenAI concepts, specifically a simplified version of Retrieval-Augmented Generation (RAG).



The system:

\- Takes user input in natural language

\- Retrieves relevant service data from a structured dataset

\- Generates a contextual response including service recommendation, professionals, and available slots



This approach demonstrates how GenAI systems combine input understanding, retrieval, and response generation, even though it is implemented in a lightweight frontend-only setup.



\---



\## Summary

AI tools were used only to speed up basic UI setup. The main application logic, design decisions, and GenAI-inspired implementation were developed and integrated by me.
